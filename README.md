# Hospital Emergency Department Flow Dashboard (Power BI)

## Project Overview
This project analyzes hospital emergency department patient flow and waiting time patterns using Power BI and Excel data.

The goal is to identify peak congestion periods, average waiting time trends, and department-wise performance.

## Tools Used
- Power BI Desktop
- Microsoft Excel

## Key Features
- Average wait time by hour, day of week, and department
- Interactive slicers for date range and department
- Bottleneck identification using conditional formatting
- KPI cards for quick insights

## Dataset
Source file: Appointment Scheduling Cleaned Data.csv  
Contains patient appointment and waiting time information.

## Key Insights
1. Peak waiting time occurs during high patient inflow hours.
2. Certain departments experience higher average waiting times.
3. Weekday trends show variation in congestion patterns.

## How To Use
1. Download the .pbix file.
2. Open in Power BI Desktop.
3. Use slicers to interact with the dashboard.
